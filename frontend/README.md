# SentinelX-AI Frontend

React + Vite frontend for the SentinelX-AI suspect detection system.

## Setup

```bash
npm install
npm run dev
```

Runs on **http://localhost:5173** by default.

## Requirements

The backend must be running on **http://localhost:8000** for the live analysis features to work.

See the [project README](../README.md) for full setup instructions.
